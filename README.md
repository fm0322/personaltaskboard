# Personal Task Board

A lightweight web-based personal task board running locally on Windows.

## Repository structure

- `src/PersonalTaskBoard` - main app project
- `src/PersonalTaskBoard.Tests` - test project
- `docs` - project documentation

## Run locally

```powershell
cd src/PersonalTaskBoard && dotnet run
```

## Run tests

```powershell
dotnet test src/PersonalTaskBoard.Tests/PersonalTaskBoard.Tests.csproj
```
